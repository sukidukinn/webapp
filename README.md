# webapp

This folder is the local root for the GitHub repository:

- `https://github.com/sukidukinn/webapp`

It contains two separate web properties:

- `sukidukinn.com/`
- `sukidukinn.blog/`

## Publishing model

### `sukidukinn.com`

- official site
- portfolio
- business card QR landing page
- work and contact entry point
- planned for Next.js-based deployment

### `sukidukinn.blog`

- activity log
- hobby posts
- experiments
- simple static website
- no Node.js required

## Suggested repository usage

- keep both sites in this one repository
- manage each site in its own folder
- connect Vercel projects by setting the correct root directory

## Vercel root directory idea

- official site project root: `sukidukinn.com`
- blog site project root: `sukidukinn.blog`
